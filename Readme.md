# AutoHub

AutoHub is a simple car marketplace website built with **React** (frontend) and **Node.js + Express** (backend) with **MongoDB** for data storage.

## Features

- Browse car listings with filters
- User authentication (register/login)
- Post, edit, and delete car listings (for logged-in users)

## Tech Stack

- Frontend: React, TailwindCSS  
- Backend: Node.js, Express, JWT authentication  
- Database: MongoDB (Atlas)
